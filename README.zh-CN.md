# 五子棋游戏

一款用 HTML、CSS 和 JavaScript 构建的简单互动五子棋游戏，可直接在浏览器中游玩。

## 关于五子棋

五子棋是一款传统棋类游戏，在 15×15 的棋盘上进行。两名玩家轮流在棋盘上放置棋子，目标是形成一条不间断的五个棋子的连线，这条线可以是水平、竖直或对角线方向。

## 功能特性

- **互动游戏**：点击任何空位放置棋子
- **双人模式**：在黑白两种颜色的棋子之间轮流操作
- **自动赢棋检测**：自动检测玩家是否获胜
- **重新开始按钮**：随时可以开始新游戏
- **响应式设计**：在桌面和移动设备上都能正常运行

## 游戏规则

1. 游戏以黑棋开始
2. 玩家轮流点击空位放置棋子
3. 第一个形成五子连线（水平、竖直或对角线）的玩家获胜
4. 点击"重新开始"按钮可开始新游戏

## 本地运行

如果您想在本地运行此项目：

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/vokako/gomoku-game.git
   ```

2. 进入项目目录
   ```bash
   cd gomoku-game
   ```

3. 用浏览器打开 `index.html` 文件即可开始游玩
