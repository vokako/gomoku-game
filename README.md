# Gomoku Game

A simple, interactive Gomoku game built with HTML, CSS, and JavaScript that can be played directly in your browser.

## About Gomoku

Gomoku is a traditional board game played on a 15×15 grid. Two players take turns placing stones on the board, with the goal of creating an unbroken line of five stones either horizontally, vertically, or diagonally.

## Features

- **Interactive Gameplay**: Click on any empty cell to place your stone
- **Two-Player Mode**: Take turns between Black and White players
- **Win Detection**: Automatically detects when a player has won
- **Reset Button**: Start a new game at any time
- **Responsive Design**: Works on desktop and mobile devices

## How to Play

1. The game starts with the Black player
2. Players take turns clicking on empty cells to place their stones
3. The first player to create a line of 5 stones wins
4. Click the "Reset Game" button to start a new game

## Deployment

## Installation

To run locally:

1. Clone the repository
2. Open `index.html` in your web browser
3. Start playing!
